Autobiografia

Meu nome é Leonardo e sempre fui apaixonado por tecnologia e criatividade. Desde pequeno desmontava brinquedos para entender como funcionavam, movido pela curiosidade de descobrir o que havia por trás de cada mecanismo. Essa vontade de explorar e aprender me acompanhou até hoje.

Com o tempo, mergulhei no mundo da programação. Comecei criando pequenos sites, experimentando com HTML e CSS, e logo me aprofundei em JavaScript e React. Cada projeto que desenvolvo é uma oportunidade de transformar ideias em realidade, e meu portfólio reflete essa jornada de evolução constante.

Além da tecnologia, tenho outras paixões que moldam minha forma de ver o mundo. Sou fascinado por ficção científica, que me inspira a imaginar futuros possíveis e universos alternativos. Também sou fã de animes, que me ensinam sobre criatividade, emoção e narrativas envolventes. E, claro, os jogos de RPG ocupam um espaço especial na minha vida, pois me permitem viver histórias épicas e desenvolver estratégias que estimulam tanto a mente quanto a imaginação.

Meu objetivo é continuar crescendo como desenvolvedor, sempre buscando unir minhas habilidades técnicas com minhas paixões pessoais. Acredito que cada linha de código é uma oportunidade de construir algo que pode impactar positivamente outras pessoas.

  • E-commerce de Comida com React

    Principais funcionalidades
    Navegação dinâmica: botões que levam diretamente a categorias específicas de comidas.
    Carrinho de compras interativo: o usuário pode adicionar e remover itens em tempo real, com atualização automática do valor total.
    Fluxo de compra completo: na conclusão da compra, é feita uma requisição a uma API para processar os pedidos.
    Cadastro de endereço e pagamento: formulários integrados para inserção de dados do cliente.
    Modal de pagamento: interface simplificada para finalizar a compra com segurança.
    Tecnologias utilizadas React para construção da interface.
    Bibliotecas do ecossistema React (como React Router para navegação e Context API/Redux para gerenciamento de estado).
    Integração com API REST para envio dos pedidos.
    Componentes reutilizáveis para botões, cards de produtos e modais.
    Objetivo O projeto demonstra habilidades em desenvolvimento front-end moderno, integração com APIs e criação de uma experiência de compra fluida.

  • Site de Filme – De Volta para o Futuro

    Este projeto é um site temático desenvolvido com HTML, CSS e JavaScript, utilizando Gulp para automação de tarefas e otimização de performance. O objetivo foi criar uma experiência semelhante a um blog de cinema, com foco no clássico De Volta para o Futuro. Principais funcionalidades
    Página inicial estilizada com design responsivo e elementos visuais que remetem ao filme.
    Links diretos para trailers e cenas no YouTube, permitindo que o usuário assista ao conteúdo sem sair do site.
    Seções com descrições detalhadas sobre o filme, curiosidades e contexto histórico, no estilo de artigos de blog.
    Navegação simples e intuitiva, com menus e botões que levam a diferentes partes do conteúdo.
    Automação com Gulp:
    Minificação de arquivos CSS e JavaScript.
    Otimização de imagens.
    Live reload para agilizar o desenvolvimento.
    Tecnologias utilizadas HTML5 para estrutura semântica.
    CSS3 para estilização responsiva e animações.
    JavaScript para interatividade, como botões e links dinâmicos.
    Gulp para automação de tarefas e melhoria de performance.
    Objetivo O projeto demonstra domínio em front-end clássico (HTML, CSS e JS) aliado ao uso de ferramentas modernas de build como Gulp. É um exemplo de como unir conteúdo cultural e tecnologia web, criando um site envolvente e otimizado para portfólio.

  • Game Shop – Loja de Games Fictícia

    Este projeto é um site fictício de e-commerce de games, desenvolvido com HTML5 e CSS3, com foco em apresentar uma interface simples, moderna e funcional.
    Principais funcionalidades
    Página inicial atrativa com destaque para os principais jogos e promoções.
    Formulário de contato com campo de input para que o usuário possa enviar mensagens ou dúvidas diretamente pela página.
    Links para redes sociais integrados no rodapé, permitindo fácil acesso às páginas da loja em plataformas como Facebook, Instagram e Twitter.
    Layout responsivo que se adapta a diferentes dispositivos (desktop, tablet e mobile).
    Estilo personalizado com CSS para cores, tipografia e organização visual, transmitindo a identidade gamer.
    Tecnologias utilizadas HTML5 para estrutura semântica e organização do conteúdo.
    CSS3 para estilização, responsividade e efeitos visuais.
    Formulário de contato com validação básica em HTML.
    Links externos para redes sociais, garantindo conectividade com o público.
    Objetivo O projeto demonstra habilidades em desenvolvimento front-end clássico, criação de layouts responsivos e integração de elementos essenciais para um site de loja online. É uma vitrine fictícia que mostra como aplicar conceitos de design e usabilidade em um contexto de e-commerce de games.

  • Space Tourism Website – Frontend Mentor Challenge

    Este projeto foi desenvolvido como parte do desafio Space Tourism do Frontend Mentor, utilizando React para criar uma aplicação moderna e responsiva. O objetivo foi reproduzir uma experiência de navegação imersiva, simulando um site de turismo espacial.
    Principais funcionalidades
    Navegação por abas: o usuário pode explorar diferentes seções como Destinos, Tripulação e Tecnologia.
    Interface responsiva: design adaptado para desktop, tablet e mobile, garantindo uma experiência consistente em qualquer dispositivo.
    Componentização com React: cada seção é construída com componentes reutilizáveis, facilitando manutenção e escalabilidade.
    Gerenciamento de estado: controle dinâmico de conteúdo exibido conforme a interação do usuário.
    Animações e transições: efeitos visuais que tornam a navegação mais envolvente.
    Tecnologias utilizadas React para construção da interface.
    React Router para navegação entre páginas e seções.
    CSS Modules / Styled Components para estilização modular e organizada.
    Arquitetura baseada em componentes para reutilização e clareza do código.
    Objetivo Este projeto demonstra habilidades em React moderno, responsividade e design orientado a desafios práticos, sendo uma excelente vitrine para portfólio. Ele mostra como transformar um protótipo do Frontend Mentor em uma aplicação funcional e interativa.

  • Site de Contagem Regressiva – Festa de Aniversário

    Este projeto é um site de contagem regressiva desenvolvido com HTML, CSS e JavaScript, pensado para criar expectativa e engajamento antes de uma festa de aniversário.
    Principais funcionalidades Contagem regressiva dinâmica: exibe dias, horas, minutos e segundos restantes até o evento.
    Atualização em tempo real: o relógio é atualizado automaticamente via JavaScript sem precisar recarregar a página.
    Design festivo e responsivo: estilização com CSS para transmitir o clima de celebração, adaptando-se a diferentes dispositivos.
    Mensagem especial ao zerar o contador: quando a contagem chega a zero, aparece uma mensagem personalizada como “🎂 Feliz Aniversário!”.
    Layout simples e intuitivo: fácil de navegar e ideal para compartilhar com convidados.
    Tecnologias utilizadas HTML5 para estrutura semântica.
    CSS3 para estilização e responsividade.
    JavaScript para lógica da contagem regressiva e manipulação do DOM.
